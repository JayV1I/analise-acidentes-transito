# analise-acidentes-transito
Projeto de Ciência de Dados para análise e predição da gravidade de acidentes de trânsito. Inclui tratamento de mais de 312 mil registros, armazenamento em PostgreSQL, análise exploratória, engenharia de atributos, Machine Learning para classificação de acidentes graves e dashboard interativo para visualização de indicadores e padrões.
